# Khushhal Mandal Portfolio

A minimal, professional portfolio starter built as a single-page site. It is designed to be easy to update later with real resume content, more projects, and extra sections.

## What is included

- Hero section with name and call-to-action buttons
- About, experience, projects, skills, education, and contact sections
- Responsive layout for desktop and mobile
- Simple structure that can be expanded later

## How to customize

- Replace placeholder project titles and descriptions in `index.html`
- Update skills, education, and contact links
- Add a resume PDF link when you are ready
- Expand the sections or split the site into multiple pages later

## Deploying to Vercel

This project is already ready for Vercel as a static site.

1. Push the folder to GitHub.
2. Import the repository in Vercel.
3. Deploy with the default settings.

You can still change the site after deployment. If the Vercel project is connected to GitHub, pushing changes to the repo will trigger a new deployment automatically. If you deploy manually, you will need to redeploy after each update.
